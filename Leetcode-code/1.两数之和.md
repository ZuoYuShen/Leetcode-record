题目：给定一个整数数组 nums 和一个目标值 target，请你在该数组中找出和为目标值的那 两个 整数，并返回他们的数组下标。
你可以假设每种输入只会对应一个答案。但是，你不能重复利用这个数组中同样的元素。  

思路1：暴力解法，两个循环变量遍历，将结果作为数组输出。  

思路2：哈希表。  

哈希表：C++中构建哈希表unordered_map<int, int> m;  
简单说，解决这一题，可以把映射关系表达为：m[nums[i]] = i  
这样，在得到nums[i]以后，就有t = target – nums[i]  
在哈希表中，寻找m[t]是否存在(m.count(t))  
同时有一种情况需要排除：例：[3,2,4] target=6。看到，6=3+3，因此如果不做对应的判断，输出就变成[0,0]。要避免这种情况，需要加上约束：m[t] != i，“防止哈希表找到自己”。
